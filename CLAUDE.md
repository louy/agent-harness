# Agent Harness

This is a Claude Code plugin marketplace owned by Louay Akkad. It hosts the `agent-harness` plugin, which provides custom skills and hooks for use across projects.

## Structure

- `.claude-plugin/marketplace.json` — Marketplace definition
- `plugins/agent-harness/` — The agent-harness plugin
  - `skills/` — Custom skills
  - `hooks/` — Hook definitions

## Maintenance Rules

- Whenever a skill, hook, or other plugin component is added, removed, or modified, update the plugin's `README.md` to reflect the change (skills table, hooks section, etc.)
